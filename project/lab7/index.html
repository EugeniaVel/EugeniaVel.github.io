Lab №7

May 14,2022

Theoretical introduction

Midnight Commander (or mc) - pseudo-graphical command shell for UNIX/Linux systems.

Above the panels is a menu that can be accessed using F9 keys. Under the panels at the bottom there are on-screen control buttons, associated with function keys F1 to F10. Above them is command line for entering commands.

The panel in mc displays a list of files in the current directory. The absolute path to this directory is displayed in the header of the panel. The active panel has a title and one of its lines are highlighted. Panels are controlled using certain keyboard shortcuts or menu items mc.

You can go to the mc panels menu bar using the F9 function key. The menu bar has five menus: Left Panel , File , Command , Options and Right Panel.

Performing the lab

  1. Learned information about Midnight Commander by calling man mc on the command line.

  1. Launched mc from the command line, studied its structure and menu.

b)Using a combination of shift keys and up and down arrows, I selected the ~/work/study directory in the right pane, then, using the same combination, deselected the directory.

c) Using the f5 key, I copied the Student.cpp file from the ~/work/study/2021-2022/Operating systems directory to the ~/work directory.

d) Using the f6 key, I moved the Student.cpp file from the ~/work directory to the home directory.

e) Using the combination of shift and space keys, I looked at the size of the home directory.

  1. Executed the commands:

a) Quick view. With its help, I displayed the contents of the Student.cpp file from the home directory on the left panel.

b) Information. With its help, I displayed information about one of the screenshots on the screen.

c) Directory tree. With its help, I looked at the contents of the home directory in the form of a directory tree.

d) List format. With its help, I changed the format of the output list of files and directories to a shortened one.

e) Sorting order. With its help, I changed the sort order of files and directories in the home directory.

f) Filter. With its help, I displayed files from the home directory only with the .cpp extension.

  1. Using the capabilities of the file submenu, performed the following operations:

-Viewed the contents of the Student.cpp file using the View File command.

-Edited the Student.cpp file using the edit command without saving the changes.

-Created the ~/lab7 directory using the create directory command.

-Using the copy command, I copied the two screenshots I selected to the ~/lab7 directory I created in the last paragraph.

  1. Using the possibilities of the submenu, the command performed the following operations:

-Recursively found all files with the .txt extension using the command search for a file in the home directory.

-With the help of the history command, I looked at the history of commands, but did not receive the output of any command, so I could not complete the task to the end.

-With the help of the quick access directories command, where I previously added the home directory, I moved to it.

-Using the command to edit the menu file, I looked at the contents of the menu file. The command allows you to edit the user’s context menu called by the F2 key. Using the edit extension file command, I looked at the contents of the extension file. The command allows you to set, using a specific syntax, actions when starting files with a specific extension.

  1. Called up the settings submenu. In the panel settings, I found the command to show hidden files. and other commands that define the structure of the mc screen. (For example, changing the theme of mc)

Task for the built-in mc editor

  1. Using the touch command, I created a text.txt file in my home directory.

  2. Opened the file in the editor using the mc -e command (you can also use the mcedit command).

  1. Inserted the text of the template for laboratory work into the file.

4.1. Using the key combination ctrl-y deleted the first line in the text of the file.

4.2. Using the f3 key, I selected a fragment of the text of the file (one paragraph) and, using the f5 key, copied it to the first line.

4.3. Using the f3 key, I selected a fragment of the text of the file (one paragraph) and, using the f6 key, moved it to the next line after the second paragraph.

4.4. Using the f2 key, save the changes to the file.

4.5. Using the key combination ctrl-u canceled the paragraph move made in subsection 4.3.

4.6. Using the key combination ctrl-end moved to the end of the file and entered her full name.

4.7. Using the key combination ctrl-home moved to the beginning of the file and entered the name of her group.

4.8. Saved the file with the f2 key and closed the file with the f10 key.

  2. Using the editor menu (command submenu), turned off the syntax highlighting, since it was originally on.
